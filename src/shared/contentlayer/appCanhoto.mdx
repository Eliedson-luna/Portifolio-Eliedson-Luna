<About>
    <Paragraph >
    O AppCanhoto foi criado para fotografar assinaturas que comprovam o recebimento de 
    cargas pelos compradores, garantindo uma maior segurança no processo de logística 
    de uma empresa.
    </Paragraph>    
</About>
<Features className="flex flex-col justify-around ">
    <Feature title='Armazenamento de Imagens'>
    Permite que as fotos das assinaturas sejam salvas diretamente no dispositivo, mesmo sem conexão com a internet.
    </Feature>

    <Feature title='Consulta de Imagens Armazenadas'>
    O usuário pode visualizar as imagens registradas, facilitando a verificação e controle dos comprovantes de entrega.
    </Feature>

    <Feature title='Envio de imagens via API'>
    As imagens podem ser enviadas posteriormente para um servidor, permitindo centralização dos dados e integração com sistemas corporativos.   
    </Feature>
</Features>
