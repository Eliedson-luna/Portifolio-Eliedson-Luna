    <Paragraph align="justify">
    Esse projeto se trata de um desafio que encontrei pelo meu caminho como desenvolvedor. Nele eu precisava criar uma interface que fazia requisições para um backend e depois exibisse as informações na tela.
    </Paragraph>
    <Paragraph align="justify">
    O Backend por sua vez era simples, devia apenas receber os parâmetros de busca, redirecionando-os para a api oficial da amazon, agindo como um intermediador de informações.
    </Paragraph>
    <Paragraph align="justify">
    Tentei ao máximo desenvolver uma interface limpa e responsiva para oferecer a melhor experiência ao usuário. Fiquei muito contente com o resultado, pois ,apesar de ser um projeto simples e com uma interface mais simples ainda, não deixou de ser uma boa fonte de experiência para mim.
    </Paragraph>
